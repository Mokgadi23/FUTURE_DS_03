# 📊 Marketing Funnel & Conversion Performance Analysis

## 📌 Project Overview
This project focuses on analyzing user behavior across a marketing funnel to understand how users move from viewing products to making purchases.

The goal is to identify drop-off points, measure conversion rates, and provide actionable insights to improve business performance.

---

## 🎯 Objectives
- Analyze user journey: View → Cart → Purchase
- Identify drop-off points in the funnel
- Evaluate conversion rates
- Provide business recommendations

---

## 🛠 Tools Used
- Power BI (Dashboard & Visualization)
- Excel / CSV (Data Source)

---

## 📊 Key Insights
- Majority of users only view products and do not proceed to purchase
- Significant drop-off between cart and purchase stages
- A small number of products and brands generate most engagement
- Peak user activity occurs during specific hours

---

## 📈 Recommendations
- Improve product page experience to increase add-to-cart rate
- Simplify checkout process to reduce drop-off
- Target high-performing products and brands for marketing campaigns
- Optimize marketing during peak activity hours
